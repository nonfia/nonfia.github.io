Thymeleaf 模板引擎

Nov 20, 2024

Thymeleaf 介绍

Thymeleaf 是新一代 Java 模板引擎，它类似于 Velocity、FreeMarker 等传统 Java 模板引擎，但是与传统 Java 模板引擎不同的是，Thymeleaf 支持 HTML 原型。

它既可以让前端工程师在浏览器中直接打开查看样式，也可以让后端工程师结合真实数据查看显示效果，同时，SpringBoot 提供了 Thymeleaf 自动化配置解决方案，因此在 SpringBoot 中使用 Thymeleaf 非常方便。

事实上， Thymeleaf 除了展示基本的 HTML ，进行页面渲染之外，也可以作为一个 HTML 片段进行渲染，例如我们在做邮件发送时，可以使用 Thymeleaf 作为邮件发送模板。

Thymeleaf 官网

引入依赖

<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-thymeleaf</artifactId>
</dependency>

自动渲染

定义 Controller， 返回逻辑视图 index，Spring Boot 会到 resources/templates 目录下寻找一个名为 index.html 的 Thymeleaf 模板文件

@GetMapping("/")
public String index(Model model) {
    model.addAttribute("name", "nonfia");
    model.addAttribute("account", "root");
    return "index";
}

在 resources/templates 创建 Thymeleaf 模板文件 index.html

<!DOCTYPE html>
<html lang="en" xmlns:th="http://www.thymeleaf.org">
<head>
    <meta charset="UTF-8">
    <title>index</title>
    <!-- js 获取数据 -->
    <script th:inline="javascript">
        let account = [[${account}]];
        console.log(account)
    </script>
</head>
<body>
    <h1>Hello!</h1>
    <p>Welcome to <span th:text="${name}"></span></p>
</body>
</html>

手动渲染

新建一个邮件模板 mail.html

<!DOCTYPE html>
<html lang="en" xmlns:th="http://www.thymeleaf.org">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
<p>hello 你好 <span th:text="${name}"></span>您的工资信息如下：</p>
<table>
    <tr>
        <td>职位</td>
        <td th:text="${position}"></td>
    </tr>
    <tr>
        <td>薪水</td>
        <td th:text="${salary}"></td>
    </tr>
</table>
</body>
</html>

后端代码处理

@Autowired
TemplateEngine templateEngine;

public void mail() throws MessagingException {
    Context context = new Context();
    context.setVariable("name", "Tom");
    context.setVariable("position", "工程师");
    context.setVariable("salary", 88888);

    // 调用 process 方法进行渲染，该方法的返回值就是渲染后的 HTML 字符串
    String mail = templateEngine.process("mail", context);
}
